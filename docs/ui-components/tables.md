# 表格

表格默认是响应式的，过宽的表格可以通过横向滚动访问窗口之外的列。

<div class="code-example" markdown="1">

| 表头一      | 表头二            | 三    |
|:-------------|:------------------|:------|
| ok           | good swedish fish | nice  |
| out of stock | good and plenty   | nice  |
| ok           | good `oreos`      | hmm   |
| ok           | good `zoute` drop | yumm  |

</div>
