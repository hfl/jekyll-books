# 标记

为您的文档章节添加标记。标记有不同颜色，默认是蓝色。

<div class="code-example" markdown="1">
默认标记
{: .label }

蓝标
{: .label .label-blue }

稳定版
{: .label .label-green }

新版
{: .label .label-purple }

开发中
{: .label .label-yellow }

已废弃
{: .label .label-red }
</div>
